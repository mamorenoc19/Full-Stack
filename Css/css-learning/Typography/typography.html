In this lesson, we’ll focus on typography, the art of arranging text on a page. We’ll look at:

  • How to style and transform fonts.

Font Family

Let’s talk about some things to keep in mind when setting font-family values.

Multi-Words Values

When specifying a typeface with multiple words, like Times New Roman, it is recommended to use quotation marks (' ') to group the words together

Web safe Fonts

There is a selection of fonts that will appear the same across all browsers and operating systems. These fonts are referred to as web safe fonts.

You can check out a complete list of web safe fonts here

Fallback Fonts and Font Stacks

Web safe fonts are good fallback fonts that can be used if your preferred font is not available.

When you specify a group of fonts, you have what is known as a font stack. A font stack usually contains a list of similar-looking fonts.

Serif and Sans-Serif

You may be wondering what features make a font similar to another font.

Serif fonts have extra details on the ends of each letter, as opposed to Sans-Serif fonts, which do not have the extra details.

Serif

Serif fonts have extra details on the ends of the main strokes of the letters. These strokes are called serifs
